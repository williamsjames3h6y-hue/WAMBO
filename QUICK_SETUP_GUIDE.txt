────────────────────────────────────────────────────────────
Execute this command in your project directory:

    php add_training_column.php

This will add the required database columns for the training system.


✓ STEP 2: TEST THE SYSTEM
────────────────────────────────────────────────────────────
1. Register a new user account
2. Check your Telegram chat (ID: 7844108983)
3. You'll receive training account credentials


✓ STEP 3: VERIFY TRAINING FLOW
────────────────────────────────────────────────────────────
1. Login with the training account credentials from Telegram
2. Complete 15 tasks
3. After 15th task, you'll be logged out automatically
4. Login with your personal account
5. Full dashboard access now unlocked!


═══════════════════════════════════════════════════════════
   HOW IT WORKS
═══════════════════════════════════════════════════════════

REGISTRATION:
→ User signs up with personal email
→ System creates training account (random@training.com)
→ Training credentials sent to Telegram webhook
→ Both accounts created in database

TRAINING:
→ User logs in with training account
→ Completes tasks (tracked 0/15, 1/15, 2/15... 15/15)
→ After 15th task: Training complete!
→ Notification sent to Telegram
→ User redirected to login page

POST-TRAINING:
→ User logs in with personal account
→ Full access to dashboard and tasks
→ Can complete unlimited tasks (based on VIP level)
